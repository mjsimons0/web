In [8]:
# Plot the average value of a feature for each quality rating
def plot_quality_avg(feature):
    avg = wine_data.groupby('quality')[feature].mean() #Find each quality rating's avg feature value
    #Produce a labeled scatter plot for average feature value vs. quality rating
    plt.figure(figsize=(6, 5))
    plt.scatter(avg.index, avg.values, color='g', s=100)
    plt.title(f'Average {feature} value for each quality rating', fontsize=15)
    plt.xlabel('Quality Rating', fontsize=13)
    plt.ylabel(f'{feature}', fontsize=13)
    plt.grid()

plot_quality_avg('volatile acidity')

In the first plot, there is a clear downward trend; as the quality rating increases, the average volatile acidity decreases. It should be noted that quality 8 is not consistent with this trend, as its average volatile acidity rating is greater than that of quality 7. Nonetheless, the general trend indicates that volatile acidity is hihly correlated with quality rating.

plot_quality_avg('sulphates')

Our second plot depicts a strong positive correlation between the sulphates value and quality, showing that higher sulphate values are likely associated with higher quality wine.

Another positive trend is shown between citric acid values and quality rating.

In [11]:

The fourth plot indicates that there is generally a trend between alcohol values and quality, however it should be noted that the rating 5 deviates from this trend.

In [12]:
