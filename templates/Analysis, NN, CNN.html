We will now define the target and input features. Since the aim is to predict someone's income class, the target will be income, and the input features will be the rest of the data. After defining the targets and inputs, the data will be split into training and testing sets.

In [13]:
targets=income_data['income'] #Define target column
inputs=income_data.drop(columns=['income','fnlwgt','education.num','workclass']) #Define input columns

split_idx=int(0.8 * income_data.shape[0]) #Compute 80% training/20% test split index

training_targets=targets[:split_idx] #80% training data
training_inputs=inputs[:split_idx]

testing_targets=targets[split_idx:]  #20% testing data
testing_inputs=inputs[split_idx:]

print(f'Training targets shape: {training_targets.shape}, Training inputs shape: {training_inputs.shape}')
print(f'Testing targets shape: {testing_targets.shape}, Testing inputs shape: {testing_inputs.shape}')

Training targets shape: (26048,), Training inputs shape: (26048, 55)

The neural network can now be built to predict the income category of an entry. We will a utilize fully-connected dense network with two hidden layers. We are using 55 input features, so the input dimension will be 55. The input layer will be of size 256, next hidden layer size 128, and the final hidden layer size 64. The output layer will be of size 1. The input layer and hidden layers will use ReLU activaton, and the output layer sigmoid activaton, as we are performing binary classification. Finally, the Adam optimizer will be used with a binary crossentropy loss function. We will begin training the model with 110 epochs, and visualize the results.

In [14]:
from tensorflow.keras import models,layers,utils,optimizers
model=models.Sequential() #Initialize the model
model.add(layers.Dense(256, activation='relu', input_dim=training_inputs.shape[1])) #input layer, ReLU
model.add(layers.Dense(128, activation='relu')) #hidden layer, ReLU activation
model.add(layers.Dense(64, activation='relu')) #hidden layer, ReLU activation
model.add(layers.Dense(1, activation='sigmoid')) #Output layer, sigmoid activation
model.compile(optimizer=optimizers.Adam(),loss='binary_crossentropy',metrics=['accuracy']) #Compile using binary crossentropy
history=model.fit(training_inputs,training_targets,validation_data=(testing_inputs,testing_targets),epochs=110,batch_size=128) #Train the model for 110 epochs

Convolutional Neural Network¶

This section will aim to construct and train a convolutional neural network (CNN) which classifies an object based upon an image. Specifically, we will utilize the fashion MNIST dataset. The dataset contains 70,000 28x28 images of fashion-related objects, each associated with a label from a set of 10 possible classes. To begin, we will extract the dataset using the convenient methods included in the keras library.

Prior to building the CNN, we will normalize the pixel values of both the training and testing images. Dividing the pixel values by 255 will scale the values from the original range of 0-255 to a range of 0-1.

In [22]:

The aim is now to design the CNN. Using the sequential model included in the keras library will enable the addition of a sequence of layers.

In [23]:
model=models.Sequential()

The CNN will be trained to classify the fashion images and predict their labels. Cross-entropy loss will be utilized for this task, and will use the keras adam optimizer. The network will begin with a short sequence of ReLU-activated convolutional layers. This choice of activation will prevent the vanishing gradient problem. The first layer will utilize 32 3x3 filters, and the second layer will utilize 64. Each filter will have a stride of 1.

In [24]:
model.add(layers.Conv2D(filters=32, kernel_size=3,activation='relu', input_shape=(28, 28, 1))) #Input layer (Conv1)
model.add(layers.Conv2D(filters=64, kernel_size=3,activation='relu')) #Conv2

Following the convolutional layers, the network will utilize several regularization techniques such as batch normalization and dropout to prevent the model from overfitting. Max pooling will be used as a sub-sampling technique prior to flattening the data.

In [25]:
model.add(layers.BatchNormalization()) #Normalization
model.add(layers.MaxPooling2D(pool_size=2)) #Sub-sampling
model.add(layers.Dropout(0.15)) #Regularization effect
model.add(layers.Flatten()) #Flatten data

Once the data has been flattened, it will pass through a fully-connected ReLU-activated layer consistng of 128 neurons. As a final regularizer, a dropout layer will be added. Past the dropout layer is the output layer, consisting of 10 neurons, each representing one of the labels 0-9. As this is a multi-class classification, softmax activation will be utilized to classify the label, returning a single prediction.

In [26]:
model.add(layers.Dense(128, activation='relu')) #Fully-conntected layer
model.add(layers.Dropout(0.15)) #Final regularizer
model.add(layers.Dense(10, activation='softmax')) #Output layer
model.compile(optimizer=optimizers.Adam(), loss='categorical_crossentropy', metrics=['accuracy']) #Compile the structure
print(model.summary()) #Display the complete architecture
